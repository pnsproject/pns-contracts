部分状态，例如 ~PNS.FLAGS~ ，在过程中可能会变化，最终应该和迁移前保持一致。

对于 ~Controller.records~ 到 ~PNS.records~ 的迁移，类型由 ~Controller.Record~ 变为 ~PNS.Record~ ，其中 ~origin~ 和 ~expire~ 域保持不变，抛弃原 ~children~ 和 ~capacity~ ，需要新增 ~parent~ 域。
